# 钓法表达式快速修改指南  

## 0. 修改前须知

- **修改现有钓法表达式可以达成以下目的：**  
  从已有的类似钓法来修改来钓不同的鱼；  
  修改成想要的钓法；  
  修正错误的钓法；  

### 0.1 获取表达式
建议从现有预设的表达式开始，有不同方式可以获取表达式：  

1. 在预设页搜索预设，右键对应预设展开快捷菜单，点击「填写到编辑器」。  
2. 从[在线预设表格](https://docs.qq.com/sheet/DVlNaQ012amF4SW1G?tab=yirja0)获取。  

### 0.2 格式化
在修改前，使用钓法页的编辑器将表达式格式化来增强可读性。  

1. 确认编辑器已填写有合法的表达式。  
2. 点击格式化，或在不选中任何文本时，Ctrl + S / Ctrl + X。  

### 0.3 使用
修改完成后，可直接在编辑器点启动按钮，来使 MissFisher 以当前编辑器的钓法自动钓鱼。  

或者点击复制以获取可在命令中使用的单行格式的表达式，在主命令后接表达式如 `/mf 平钓》！！！`。也可保存到游戏的用户宏中。  

但注意：表达式超长时，发送时可能被游戏截断，而导致钓法异常或无法识别。  

## 1. 修改目标
表达式中有两种目标：  
不带数字的是终止目标，如 `【红龙】；`；  
带数字的是计数器，如 `1【苦尔鳗】；`。  

目标通常在 `=` 后，且被 `【】` 包围。  

### 1.1 使 MF 不停止

例如目标是 `罗敏萨鳀鱼`：  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
-     =【罗敏萨鳀鱼|4870】；  
+ 平钓【沙蚕|2585】  
+     》！  
```  

是计数器的情况：  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
-     =1【罗敏萨鳀鱼|4870】；  
+ 平钓【沙蚕|2585】  
+     》！  
```  

后面有修饰词的情况：  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
-     =【罗敏萨鳀鱼|4870】；  
-         不撒饵；  
+ 平钓【沙蚕|2585】  
+     》！  
+     =不撒饵；  
```  

### 1.2 使 MF 在钓到特定目标后停止

注意要使用对应客户端语言的名称，比如在中国服客户端就应该使用 `梅尔托尔虾虎`，使用 ID 也可以，例如 `4869`。同时存在名和 ID 时， MissFisher 会优先使用 ID。  

- 这些都有效： `梅尔托尔虾虎`、`4869`、`梅尔托尔虾虎|4869`  

把 `罗敏萨鳀鱼` 改成 `梅尔托尔虾虎`：  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
-     =【罗敏萨鳀鱼|4870】；  
+ 平钓【沙蚕|2585】  
+     》！  
+     =【梅尔托尔虾虎】；  
```  

同时指定 `罗敏萨鳀鱼` 和 `梅尔托尔虾虎`，使用 `、` 来列举：  
```diff  
- 平钓【沙蚕|2585】  
-     》！  
-     =【罗敏萨鳀鱼|4870】；  
+ 平钓【沙蚕|2585】  
+     》！  
+     =【罗敏萨鳀鱼|4870、梅尔托尔虾虎】；  
```  


改成钓特定数量：  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
-     =【罗敏萨鳀鱼|4870】；  
+ 平钓【沙蚕|2585】  
+     》！  
+     =5【罗敏萨鳀鱼|4870】；  
```  

## 2. 修改提钩

### 2.1 让 MissFisher 对不同类型的咬饵使用提钩

表示咬饵类型的符号：`！` / `全部`  

例如把轻竿修改成鱼王竿：  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
+ 平钓【沙蚕|2585】  
+     》！！！  
```  

又如同时对多种咬饵类型使用提钩：  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
+ 平钓【沙蚕|2585】  
+     》！+！！！  
```  

用 `全部` 对所有类型都使用提钩：  
```diff  
- 平钓【沙蚕|2585】  
-     》！  
+ 平钓【沙蚕|2585】  
+     》全部  
```  

### 2.2 让 MF 对特定参数的咬饵使用指定提钩

在表示咬饵类型的符号后添加提钩关键字：  
- `双提` / `双重`  
- `三提` / `三重`  
- `精准` / `强力`  

例如在轻竿时使用三重提钩：  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
+ 平钓【沙蚕|2585】  
+     》！三提  
```  

## 3. 咬饵时间

### 3.1 使 MF 对特定时间的咬饵使用提钩

表示时间的数字和符号：  

| 连接数字的符号   | 相对符号的位置 | 含义        | 示例       |  
|-----------|---------|-----------|----------|  
| `~` / `-` | 前       | 最短咬饵时间    | `10~`    |  
| `~` / `-` | 后       | 最长咬饵时间    | `~10`    |  
| `+`       | 前       | 有撒饵时的咬饵时间 | `10+...` |  
| `+`       | 后       | 无撒饵时的咬饵时间 | `...+10` |  

在咬饵符号前添加咬饵时间，下面的例子表示10秒以下的咬饵：  

可只添加无撒饵时的咬饵时间，在编辑器格式化后自动补全推测的有撒饵的咬饵时间。  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
+ 平钓【沙蚕|2585】  
+     》~11！  
```  

可能会补全为：  
```diff  
- 平钓【沙蚕|2585】  
-     》~11！  
+ 平钓【沙蚕|2585】  
+     》~5+11！  
```  


更复杂的例子：  
```diff  
- 平钓【沙蚕|2585】  
-     》！  
+ 平钓【沙蚕|2585】  
+     》5+11~20+42！  
```  

## 4. 拍击水面、专一垂钓和额外提钩组

### 使 MF 对特定鱼使用拍击水面

在主目标的咬饵类型符号后使用 `@` 然后 `拍水` 再写名。  

比如拍 `梅尔托尔虾虎`：  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
-  
+ 平钓【沙蚕|2585】  
+     》！  
+         @拍水【梅尔托尔虾虎】  
```  

### 使 MF 在需要拍击水面时对特定咬饵使用提钩

在 `拍水` 后添加咬饵参数：  
这样会在没有拍击水面时额外对 `！` 使用提钩。  

```diff  
- 平钓【沙蚕|2585】  
-     》！！！  
-         @拍水【梅尔托尔虾虎】  
+ 平钓【沙蚕|2585】  
+     》！！！  
+         @拍水！【梅尔托尔虾虎】  
```  

### 使 MF 对特定鱼使用专一垂钓

和拍击水面类似， 在主目标的咬饵类型符号后使用 `@` 然后 `专一` 再写名。  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
-  
+ 平钓【沙蚕|2585】  
+     》！  
+        @专一【罗敏萨鳀鱼】  
```  

和拍水连用：  
```diff  
- 平钓【沙蚕|2585】  
-     》！  
-        @专一【罗敏萨鳀鱼】  
-  
+ 平钓【沙蚕|2585】  
+     》！  
+        @拍水【梅尔托尔虾虎】  
+        @专一【罗敏萨鳀鱼】  
```  

改变专一垂钓后的提钩：  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
-        @专一【罗敏萨鳀鱼】  
+ 平钓【沙蚕|2585】  
+     》！  
+        @专一【罗敏萨鳀鱼】三提  
```  

### 使 MF 对更多特定咬饵使用提钩

在主目标的咬饵类型符号后添加阶段顶层 `||`，再写咬饵参数。  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
-  
+ 平钓【沙蚕|2585】  
+     》！  
+        ||！！  
```  

## 5. 窗口期

在第一个 `》` 前添加 `@` 然后接着写窗口期  

添加艾欧泽亚时间窗口期：  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
+ 平钓【沙蚕|2585】@0000-0100  
+     》！  
```  

单纯添加天气窗口期：  
```diff  
- 平钓【沙蚕|2585】  
-     》！  
+ 平钓【沙蚕|2585】@（晴朗）  
+     》！  
```  

添加窗口期前的天气条件  

单纯添加天气窗口期，比如要求之前是 `小雨`：  
```diff  
- 平钓【沙蚕|2585】@（晴朗）  
-     》！  
+ 平钓【沙蚕|2585】@（小雨=》晴朗）  
+     》！  
```  

连用时间和天气：  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
+ 平钓【沙蚕|2585】@0000-0100（小雨=》晴朗）  
+     》！  
```  

## 6. 撒饵和修饰词

在末尾的 `=` 后添加修饰词。  

| 常用修饰词 | 含义          |  
|-------|-------------|  
| `不撒饵` | 不使用撒饵       |  
| `引诱`  | 自动推断并使用引诱技能 |  
| `谦逊`  | 使用谦逊之饵      |  
| `雄心`  | 使用雄心之饵      |  

如添加：  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
-  
+ 平钓【沙蚕|2585】  
+     》！  
+     =不撒饵；  
```  

有目标时，直接追加在目标后：  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
-     =【罗敏萨鳀鱼|4870】；  
-  
+ 平钓【沙蚕|2585】  
+     》！  
+     =【罗敏萨鳀鱼|4870】；  
+         不撒饵；  
```  

追加多个修饰词，用 `、` 列举：  

```diff  
- 平钓【沙蚕|2585】  
-     》！  
-     =【罗敏萨鳀鱼|4870】；  
-  
+ 平钓【沙蚕|2585】  
+     》！  
+     =【罗敏萨鳀鱼|4870】；  
+         不撒饵、引诱；  
```
