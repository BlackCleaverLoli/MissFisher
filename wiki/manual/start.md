# 开始使用
MissFisher 是一个简洁的钓鱼辅助插件，不适合纯萌新使用。  

## 能清单式全自动吗？
不能，只在部分场景下可短期多轮次自动，通常只能在单轮次内自动。  

你还必须自己准备背包空间、修装备和购买钓饵，自己在合适的窗口期或时机启动。  

## 能干什么？

### 钓特定鱼类
必须指定预设名，类脚本的有限自动钓鱼。  

- **[钓场之王、海钓中的普通海域稀有鱼、宠物、部分捕鱼人的特殊交付需求](./scenarios/fish)**  
（不包括老主顾交易）  

### 以模式按场景自动
可自动调用预设，有额外的检测和辅助行为。  

- **[出海垂钓](./scenarios/ocean)：**  
可以进行单轮全自动以完成渔分成就、收集蓝色稀有度鱼类、完成种类数量要求的奖励成就，也可使用耐力模式进行有限的多轮全自动。（不包含精确的图鉴收集）  

- **[宇宙探索](./scenarios/cosmic)：**  
可以随任务接取自动启动适合当前任务的预设（提供有可设置项），不包括自动刷任务/寻路到任务钓场。  

- **[云冠群岛](./scenarios/diadem)：**  
可以完成数个相关成就。（不包含精确的图鉴收集）  

## 做不到的

- 不能高效练级  
- 不能无脑帮你点亮图鉴  
- 没有自动修理  
- 没有自动精炼  
- 没有自动精选  
- 没有自动交易大地票据  
- 没有自动购买钓饵  
- 不能自动消除钓场警惕  

## 相关插件依赖

MissFisher 依赖部分插件来提供额外功能。  

可在主界面 - 标题栏的齿轮设置按钮打开设置页，找到插件依赖部分查看还未安装或启动的依赖插件。  

|      依赖的插件       | MissFisher 用它来做什么         |  
|:----------------:|---------------------------|  
|  Daily Routines  | 借用它的 Edge TTS 进行语音信息播报和提醒 |  
|     vnavmesh     | 它可以提供地图内的自动寻路 / 导航        |  
|       ICE        | 自动接取、刷取、提交宇宙探索任务          |
