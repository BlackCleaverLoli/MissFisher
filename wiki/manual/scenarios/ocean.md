# 出海垂钓

## 模式

| 模式     | 主要目的                | 备注                                                                                  |
|--------|---------------------|-------------------------------------------------------------------------------------|
| *冲分*   | 称号【海王】【世界级钓手】       | 渔分的高低比较看运气                                                                          |
| *蓝鱼*   | 点亮蓝色稀有度鱼类的图鉴        | 会尝试钓航线中所有可钓的蓝鱼，当已无蓝鱼可钓时将使用冲分手法                                                      |
| *种类x*  | 航线预报中角标x对应的数量类成就&称号 | 请留意选择的模式<br>部分数量类成就需要组队才能完成                                                         |
| *练级*   | 在出海垂钓中提升等级          | 如果你已完成 6.x 版本的主线任务且喜欢折腾<br>请尝试 [在宇宙探索练级](https://nga.178.com/read.php?tid=45353336) |
| *耐力xx* | 全自动地完成海钓循环          | 现支持：冲分/蓝鱼/练级                                                                        | 

## 启动

切换到带 `耐力` 前缀的模式，点击启动按钮即可。

## 航线预报

你可以在主界面选择 `航线` 页签查看航线预报。

一组航线预报由 2 行内容组成：

```
11-02 02:00　「索蒂斯（火萤）」「依拉丝莫龙（重铁板钩）」　梅尔托尔海峡南(昏)》加拉迪翁湾(夜)》梅尔托尔海峡北(日)　04
　　　　　　　「四海大宝贝¹」「塔尼瓦（青花鱼块）」　　　　 妖歌海海面(日)》黄金港近海(昏)》无二江下游(夜)
```

- 第 1 行是近海航线，第 2 行是远海航线。
- 每列从左到右分别是：开放登记时间、航线标识、海域与时间带、在 MissFisher 的内部序号。

### 航线预报中的航线标识

航线预报中的航线标识灵感和顺序来自 [鱼糕-出海垂钓航线](https://fish.ffmomola.com/ng/#/oceanFishing)。

| 类型              | 标识                                                                                              | 备注                                                              |
|-----------------|-------------------------------------------------------------------------------------------------|-----------------------------------------------------------------|
| 渔分              | 「挑战高分」                                                                                          | 渔分上限较高，也适合获取「海王」称号&成就；<br>全自动下少数情况能获得 3~4 万分的高分；<br>运气差时可能只有几千。 |
| 远洋<br>渔分        | 「深洋之劫」                                                                                          | 适合获取「深洋之劫」称号&成就的远洋航线                                            |
| 渔分              | 「世界级钓手」                                                                                         | 分数下限较高；<br>适合获取「世界级钓手」称号&成就的航线。                                 |
| 近海<br>蓝鱼        | 「盾齿龙」<br>「海洋蟾蜍（潮虫）」<br>「依拉丝莫龙（重铁板钩）」<br>「哈弗古法（乌贼丝）」<br>「珊瑚蝠鲼（小虾肉笼）」<br>「索蒂斯（火萤）」<br>「石骨鱼（沟鼠尾巴）」 | 圆括号中是需要额外携带的钓饵                                                  |
| 远洋<br>蓝鱼        | 「琉璃龙」<br>「狱之蟹（乌贼丝）」<br>「塔尼瓦（青花鱼块）」<br>「梅泉宝龟（石蝇幼虫）」                                              | 圆括号中是需要额外携带的钓饵                                                  |
| 近海<br>数量类<br>成就 | 「水母狂魔¹」<br>「横路不通¹」<br>「只有我最鳐摆²」<br>「捕鲨人¹」<br>「龙马惊神¹」<br>「只有我最鳐摆¹」<br>「气鲀四海¹」<br>「八爪旅人¹」         | 角标 `¹` `²`：<br>表示需求为该标识时应使用的种类模式<br>`¹` = 种类1<br>`²` = 种类2      |
| 远洋<br>数量类<br>成就 | 「大虾可重新来过²」<br>「大虾可重新来过¹」<br>「四海大宝贝¹」<br>「鱿型中队精英¹」                                               | 角标 `¹` `²`：<br>表示需求为该标识时应使用的种类模式<br>`¹` = 种类1<br>`²` = 种类2      |

## 特定鱼类

- 非幻海流的绿色稀有度鱼类，需要你在对应海域和时机手动启动对应预设。
- 具体目标可继续看 [钓特定鱼类](./fish)。
