# 云冠群岛

MissFisher 有一些模式可帮助达成相关成就：

| 空岛成就名|对应的模式|所需的钓饵 |
|:-:|:-:|-|
| 新巢居住区的捕鱼人|空岛二期|云冠浮游虫 |
| 落羽商业区的捕鱼人|空岛三期|云冠大蚊 |
| 升歌文化区的捕鱼人|空岛四期|云冠大蚊、云冠气球虫 |
| 天穹街的捕鱼人|空岛技巧点|云冠大蚊、云冠气球虫 |
| 观天的捕鱼人|空岛绿鱼|万能拟饵、云冠大蚊、云冠气球虫、云冠红气球虫 |

> [!TIP]
> - 建议的捕鱼人伊修加德重建成就完成顺序：特供绿鱼 → 二期特供 & 三期特供 → 技巧点/四期特供

## 启动

1. 在 MissFisher 主界面选择空岛相关模式。
2. 前往云冠群岛。
3. 到达钓场后启动对应模式。
