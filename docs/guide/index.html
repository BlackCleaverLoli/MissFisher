指南 ​

这一部分面向想尽快上手 MissFisher 的读者，重点是“先能用，再逐步理解”。

你应该先看哪篇 ​

  • 想直接改现成配置、尽快开始使用：先看 快速修改指南
  • 想系统理解表达式怎么写、为什么这样写：看 学习钓法表达式

推荐阅读路线 ​

  1. 先读 快速修改指南，熟悉表达式在实际使用中的修改方式。
  2. 再读 学习钓法表达式，补齐语义、结构和常见组合的理解。

这一部分能帮你解决什么 ​

  • 看懂现有钓法表达式大致在做什么
  • 在不从零重写的前提下，修改钓饵、目标或部分行为
  • 建立对 Angex / 钓法表达式的整体认识，方便继续查阅参考文档

什么时候去看“参考” ​

如果你已经会用表达式，但需要核对精确语法、关键字规则或机制细节，请转到 参考。
