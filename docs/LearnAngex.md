# 钓法表达式（Angle Expression）

## 什么是钓法表达式？

钓法表达式是一种专门设计给自动钓鱼插件 MissFisher（下称 MF）的文本指令。它由一系列高度抽象化的关键字、符号和游戏内物品名称或 ID 组成。旨在描述特定场景下特定需求的钓鱼策略，用来快速配置并启动 MF 的自动钓鱼功能。
