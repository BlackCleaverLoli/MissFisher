# 诱饵技能
诱饵技能有两种：雄心之饵和谦逊之饵，在此不重复说明其游戏中的信息。

诱饵技能的机制分两部分，一是se设计的游戏机制，另一部分是由于游戏机制和通信而造成的副作用。
## 游戏机制
- **权重与锁定效果：**
  - 诱饵技能使用后会获得状态效果
    - 每层提升一定的对应体型咬饵概率。（可视作往池子中放入更多的该类型鱼）
    - 每层提升的咬饵权重都会比前一层要高。第三层最高。
    - 每次获得状态效果都有概率触发体型锁定效果，使对应体型的鱼类必定咬饵。
      - 在获得锁定效果后再次使用，将失去锁定效果。
- **鱼影：**
  - 某些鱼类的咬饵需要使用对应体型的诱饵技能，直到触发特殊的鱼影提示词。
    - 诱饵技能的层数越高，触发的概率越大。
    - 触发时，对应鱼类的咬饵概率相比其他情况时最高。
      - 如果再次使用将不会是最高概率，但保留已触发的状态。
## 副作用
### 咬饵时间
- **咬饵真空期**
  - 诱饵技能使用后将立刻进入咬饵真空期，在这段时间内，客户端不会收到任何咬饵。
    - 咬饵真空期由 2 段时间组成，不定的通信期和固定的咬饵延后期。
      - 由于通信和服务器处理需要时间，从使用到服务器返回诱饵结果（黑底漂浮提示文本）约占咬饵真空期 0.5~2 秒。
      - 在收到诱饵结果后处于咬饵延后期，这段时间固定占 3.5 秒。

### 要二十件

由于网络通信和服务器运算需要时间，诱饵技能的咬饵结果不是立刻生效的，这会直接影响到鱼类的咬饵时间。
当诱饵被使用，会按顺序发生这些事件：
用户使用→客户端发送技能→服务器返回Buff→服务器返回诱饵结果（消息）→服务器返回咬饵

从使用到返回诱饵结果，这段时间可被称为通信期。通信期的持续时间不定，约在 0.5 ~ 2 秒，取决网络状况等因素，在这段时间内，客户端不会收到任何咬饵。

从客户端接收到诱饵结果开始算的 3.5 秒，这段时间被称为咬饵真空期。咬饵真空期固定 3.5 秒，在这段时间内，客户端不会收到任何咬饵。



当成功使用诱饵技能后，接下来的数秒内客户端将不会收到任何咬饵。
最短的咬饵会出现在客户端收到诱饵结果消息后第3.5秒。
