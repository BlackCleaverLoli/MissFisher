钓特定鱼类 ​

钓场之王、海钓中的普通海域稀有鱼、宠物、部分捕鱼人的特殊交付需求。

通用方法 ​

通过预设启动。

获取目标鱼类的信息 ​

使用各种工具查看目标鱼的窗口期或可钓条件：

  • 鱼糕
  • Fish Tracker App
  • Dalamud 插件 GatherBuddy Reborn

搜索和启动 ​

搜索：

  1. 在主界面 - 主页的搜索框搜索预设。（也可输入汉语拼音、逐字的首字母，或分词式搜索）
  2. 点击在搜索框下方的补全菜单中的预设。（边框的颜色也会提示是否有匹配的预设名）

查看、前往和启动：

  1. 点查看按钮，可在聊天窗口查看预设的备注、目标所需的传承录、获得力、钓饵。
  2. 点前往按钮，可发起前往目标所在地的自动传送和导航。
  3. 确认模式下拉显示的是预设模式。
  4. 靠近可钓区域，点启动按钮来启动。

有咬饵条件的一般鱼类 ​

可在窗口期开始前启动对应预设，MissFisher 会自动进行窗口期前的预备动作。

窗口期的开始、结束、以及钓上目标时，MissFisher 都会进行提醒。

钓场之皇 ​

部分钓场之皇有着复杂的捕鱼人之识触发条件，由于有着各种意外情况，MissFisher 难以在这种情形下完成全自动的准备和钓鱼，所以请务必查看对应钓场之皇的预设说明，从各种工具或攻略主动了解并理解钓场之皇的钓法。

鱼眼预设 ​

鱼眼预设一般以鱼名开头，或以 鱼眼 / -Fish Eyes 结尾。

MissFisher 中已有少部分鱼眼类型预设，可在符合天气条件时启动。

出海垂钓中的稀有鱼类 ​

在此指非幻海流钓场的绿色稀有度鱼类（普海绿鱼），如果你想钓蓝色稀有度的鱼类，请看出海垂钓。

流程与通用方法类似，但必须在合适的时机、海域手动启动对应预设。
