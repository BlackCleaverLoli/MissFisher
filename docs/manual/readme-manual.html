简介 ​

MissFisher 是一个基于 Dalamud 的 FF14 插件，致力于（半）自动化游戏中的钓鱼相关玩法。

CAUTION

  • MissFisher 完全免费，从未委托过任何人在任何渠道进行售卖，也从未委托过任何人以帮忙安装、更新等名义收取任何形式的费用。

特性与功能 ​

  • 支持多种钓鱼玩法：
    钓场之王、出海垂钓、宇宙探索、云冠群岛
  • 提供钓鱼辅助功能：
    提钩结果信息、抛竿计时器、咬饵播报、幻海流计时等
  • 内置优异的钓鱼手法：
    无需复杂调参，准备好钓鱼，一键启动。
  • 不抢夺用户操作：
    收竿或中断，自动行为立刻终止。

支持开发 ​

反馈 ​

你可以前往 MissFisher 的 Discord（你可以在安装后从主界面的信息页上找到入口）、或在本仓库的 Issues 提交反馈信息。

安装指南 ​

环境依赖 ​

部署 Dalamud ​

Dalamud 是一个完全免费的 FF14 插件平台。
你需要从 XIVLauncher 启动游戏，才能使用 Dalamud。

XIVLauncher CN (Soil) ​

XIVLauncher 是比官方启动器更便捷的第三方 FF14 启动器。

  • Soil 版本是什么？
    XIVLauncher CN (Soil) 是 XIVLauncher CN（蓝月）的分支版本，社区里通常称它为“橙月”。

  • 为什么更推荐使用橙月？

    • 游戏更新后，橙月通常能更快提供对 Dalamud 的适配，让你更早用上插件。
    • 中国服社区相当一部分主流插件基于橙月内核开发，因此能获得更好的兼容性与支持。
    • 另外，橙月移除了主库存在的部分不必要限制，并优化了整体逻辑。

安装并打开 XIVLauncher CN (Soil) ​

  1. 下载 XIVLauncher CN (Soil).zip
  2. 解压下载好的 XIVLauncherCN-win-Portable.zip 中的文件到合适的位置。
  3. 打开解压后路径下的 XIVLauncherCN.exe

设置 XIVLauncher CN (Soil) 并用它来启动游戏 ​

  1. 点击 XIVLauncher CN (Soil) 界面右下角齿轮图标
  2. 设置游戏文件夹路径
  3. 选择 Dalamud 页签 - 确认已勾选 启用 Dalamud
  4. 点击右下角的 提钩 按钮来保存设置
  5. 从 XIVLauncher CN (Soil) 登陆并启动游戏，Dalamud 应会自动注入游戏进程

安装 MissFisher ​

以下指引是基于 XIVLauncher CN (Soil) 的，可能与其他版本略有不同，请留意

打开 Dalamud 插件安装器 ​

已登陆角色时：

  1. 在游戏中按下键盘上的 Esc
  2. 点击 Dalamud插件

在标题画面时：

  1. 在游戏标题画面左侧找到 Dalamud 图标
  2. 移动鼠标到图标上 - 点击 插件安装器

添加第三方仓库链接并安装 MissFisher ​

  1. 点击插件安装器下方的 设置
  2. 选择 插件 页签
  3. 在 插件 页面再次点击 第三方插件 页签
  4. 在空的输入框中填写 MissFisher 的仓库链接：
https://raw.githubusercontent.com/BlackCleaverLoli/MissFisher/refs/heads/main/MissFisher.json
  1. 点击 ＋ 以添加
  2. 点击保存按钮（软盘图标）
  3. 返回插件安装器，搜索 MissFisher
  4. 点击安装并等待完成

完成认证以获得 MissFisher 使用权 ​

NOTE

认证完全免费。

CAUTION

MissFisher 开发者从未委托任何人在任何渠道以帮忙认证（验证）的理由收取费用。

识别凭证 ​

识别凭证会在 MissFisher 主界面的信息页中自动生成。

识别凭证有两种：

  • M 开头的设备识别凭证（机器码）：
    对应当前的设备信息。
    使用机器码认证后，任何角色都可在该设备上使用 MissFisher。
  • S 开头的账号识别凭证（账号码）：
    对应当前登陆的账号。
    使用账号码认证后，该账号码对应账号下的任何角色都可使用 MissFisher。

TIP

设备识别凭证的变化：

  • 在不同的设备上，会生成不同的设备识别凭证。
  • 另外，设备信息变化时，也将导致设备识别凭证发生变化。
    • 导致变化的常见情形：
      更换了不同的电脑
      更换了组成电脑的重要配件
      使用一些手段变更了配件在软件上的识别信息

认证步骤 ​

前往 Discord：

  1. 在游戏中输入 /mf 以打开 MissFisher 主界面
  2. 点击 信息 页签以展开信息页
  3. 点击 Discord 按钮来前往 MissFisher 的 Discord 服务器
  4. 按指引接受邀请并登陆或注册 Discord

在 Discord 认证：

  1. 进入「点这里认证」频道
  2. 点击聊天输入框左侧的 ＋
  3. 选择 使用APP
  4. 在命令列表中找到并点击 /verify
  5. 回到游戏中 - 在 MissFisher 主界面的信息页中，点击想要认证的任一识别凭证以将其复制到剪贴板
  6. 将识别凭证粘贴到命令参数输入框中
  7. 发送并等待机器人确认

替换凭证 ​

你可以在 MissFisher 的 Discord 服务器中的「点这里认证」频道中使用 /replace 命令来将旧凭证替换为新凭证：

获取指引与旧凭证：

  1. 进入 MissFisher 的 Discord 服务器中的「点这里认证」频道
  2. 点击输入框左侧的 ＋
  3. 选择 使用APP
  4. 在命令列表中找到并点击 /replace
  5. 回到游戏中 - 在 MissFisher 主界面的信息页中，点击想要认证的任一识别凭证以将其复制到剪贴板
  6. 将识别凭证粘贴到命令参数输入框中 - 直接发送以获取指引

用新凭证替换旧凭证：

  1. 点击输入框左侧的 ＋
  2. 选择 使用APP
  3. 在命令列表中找到并点击 /replace
  4. 将识别凭证粘贴到命令参数输入框中
  5. 点击补全列表中的 old_cert_id
  6. 复制机器人回复的指引中想要替换掉的识别凭证 - 粘贴到第 2 个命令参数输入框中
  7. 发送并等待机器人响应

临时免验凭证 ​

临时免验凭证是为了应对验证服务器的状态波动而准备的。 在发生验证服务器状态波动而导致服务断链时，作者会在 Discord 发布临时免验凭证（不能保证在波动后立刻发布）。

它会发布后数天内过期，且通常只适用于最新版本。

使用临时免验凭证

  1. 在 Discord 的公告频道复制临时免验凭证
  2. 打开 MissFisher 主界面
  3. 粘贴在临时免验凭证的输入框内，通常会立刻生效

使用说明 ​

NOTE

  • MissFisher 只能钓鱼：
    MissFisher 不能自动刺鱼，也不能帮你完全自动地一键完成所有钓鱼目标，无法独自进行 24 小时全天候挂机并处理所有情况。
  • MissFisher 不是万能的：
    不能教你学会钓鱼，也不能控制你从小白变成高手。是的 MissFisher 不会脑控你。
  • 内容不是新手攻略：
    本文假设你已对游戏中的系统和钓鱼玩法有基本了解。

如果你完全不懂钓鱼，这里有一些有用的链接：

    • 灰机 Wiki 的捕鱼人页面

  • NGA：

MissFisher 的预设 ​

IMPORTANT

  • 预设是 MissFisher 实现自动钓鱼的关键：
    在启动任何自动模式或预设前，你应该确保预设已经同步到最新版。
    你可以在主界面的设置页找到 同步 按钮，点击它即可开始将预设从本仓库同步到本地。
  • MissFisher 会自动同步预设：
    若有以下情况，MissFisher 将自动同步预设：
    • 新版本的 MissFisher 初始化时
    • 初始化时与上次自动同步相隔 1 天时

MissFisher 的预设由特殊的钓法表达式组成，目前仅由作者一人维护：腾讯表格-在线预设表格

你也可以在主界面点击预设页签来切换到预设页，使用预设页下的钓法表达式生成器来图形化地编写一条钓法表达式。

修改预设中的参数或钓法（暂缺校订）：钓法表达式快速修改指南。

如果你是经验老到的捕鱼人且发现自动时的钓法与理想钓法不符，欢迎反馈。

用 MissFisher 钓特定的鱼类 ​

NOTE

  • MissFisher 不能帮你一键收集所有图鉴
    你只能自己查看攻略网站或相关插件来确认鱼类所需条件的窗口期、自己准备钓饵和管理背包空间。
  • 别浪费自己的时间
    请在捕鱼人升到满级，且在完成职业任务以学习技能 耐心II 后，再尝试钓钓场之王（鱼王）。

准备好钓鱼： ​

  1. 访问 鱼糕 或一些攻略，查看要钓的鱼类的窗口期等信息
  2. 复制要钓的鱼类名称，将其填入 MissFisher 主界面的主页右下角的预设名输入框
  3. 点击查看按钮，MissFisher 会在消息窗口显示所需钓饵
  4. 准备好所需钓饵和至少 5 ~ 10 格背包空间

前往钓场： ​

  1. 按住 Ctrl 对预设名输入框按下鼠标滚轮（或点击前往钓场，如有），如果当前不在对应钓场的地图，MissFisher 会发起传送，然后调用其他插件来自动前往钓场
  2. 操控角色靠近钓场的可钓地面边缘，使角色面朝钓场可垂钓区域直到技能抛竿亮起

启动！ ​

  1. 对预设名输入框双击鼠标左键，或切换到预设模式点击启动按钮，MissFisher 会启动可用的对应预设

TIP

  • 对于有窗口期限制的鱼类：
    除非预设的备注中有特殊说明，否则可在窗口期到来前启动， MissFisher 将自动准备相应技能资源和状态效果。

鱼眼预设 ​

MissFisher 有部分鱼类的使用特殊技能鱼眼的预设，一般而言它们以鱼名开头并以 鱼眼 或 -Fish Eyes 结尾以表示这是鱼眼预设。

TIP

  • 鱼眼的游戏机制请参考：
    灰机 Wiki-捕鱼人-天气、时间与鱼眼

用 MissFisher 出海垂钓 ​

TIP

如果你不知道出海垂钓是什么，请看 灰机 Wiki-出海垂钓

你可使用 MissFisher 中的下列模式来自动出海垂钓：

模式主要目的备注
冲分称号【海王】【世界级钓手】渔分的高低比较看运气
蓝鱼点亮蓝色稀有度鱼类的图鉴会尝试钓航线中所有可钓的蓝鱼，当已无蓝鱼可钓时将使用冲分手法
种类x航线预报中角标x对应的数量类成就&称号请留意选择的模式

TIP

  • MissFisher 内置了完备的自动预设切换：
    启动后会全自动完成整场出海垂钓任务。且会按情况灵活地变化手法以达成目的需求。

使用 MissFisher 出海垂钓的步骤： ​

点启动就行： ​

切换到带 耐力 前缀的模式，点击启动按钮即可。

如果运行所需的条件不符合，MissFisher 将提示所需的条件。请手动满足可能未满足的运行所需条件，然后再次点击启动按钮。

TIP

  • 自由地查看想去的航线：
    选择主界面的航线页签来查看航线预报；或者 访问鱼糕

IMPORTANT

  • 使用对应的种类模式：
    选择好不同的种类模式后，将鼠标移动到启动按钮上查看悬浮提示上的对应标签
  • 种类模式对应的数量类成就：
    在出海垂钓相关的成就中，有一类数量类成就及其奖励称号，必须在单场出海垂钓任务中单人或组队钓到指定数量的同分类鱼类时，才能获得。
    • 例如成就「气鲀四海」，要求小队全体合计钓到250只以上的鲀类
    • 例如成就「大虾可重新来过」，要求个人合计钓到50只以上的虾类
    • 其他详见 灰机Wiki的相关章节

钓出海垂钓中的特定鱼类 ​

NOTE

  • MissFisher 没有如“图鉴模式”的功能
    因此，对于幻海流时的非蓝色稀有度鱼类，你只能自己手动尝试钓到。
  • MissFisher 有非幻海流时绿色稀有度鱼类的预设
    但不能自动识别并切换到对应钓法，你必须手动在对应鱼类的出现海域，并在的合适时机启动对应的预设。

钓非幻海流时的绿色稀有度鱼类 ​

你可以在主界面主页的右下角找到预设输入框，将非幻海流时的绿色稀有度鱼类名称输入，即可启动预设。

TIP

  • 在正确的航线、海域以及时机启动预设
    你应该注意钓到鱼类所需的条件，并不是所有时间的任何海域都能钓到特定鱼类。

用 MissFisher 完成宇宙探索任务 ​

在使用捕鱼人接取宇宙探索任务的数秒后，MissFisher 将自动识别任务并启动对应预设。

TIP

  • 建议安装的插件：ICE
    ICE 可以自动刷取、接取和提交特定任务。
    如果你还未安装，你可以在 MissFisher 的设置页中找到 ICE 的仓库链接。

使用云冠群岛模式的步骤 ​

前往云冠群岛 ​

  1. 准备好至少 6 ~ 10 格背包空间
  2. 在 MissFisher 主界面主页选择好空岛相关模式
  3. 前往云冠群岛

启动空岛模式 ​

  1. 启动空岛模式时，MissFisher 会尝试检查是否在钓场范围内，如果否，将尝试前往对应钓场
  2. 当角色处在对应钓场时，操控角色靠近空岛边缘，使角色面朝空岛可钓鱼区域直到技能抛竿可用
  3. 点击启动按钮

用 MissFisher 练级？ ​

NOTE

  • 缺乏特别支持：
    练级过程只占游戏中钓鱼玩法的很少一部分，且难以重复测试效果。
    因此，MissFisher 的自动行为并没有专门针对未满级的捕鱼人或练级过程进行完全的优化。

你可以在主界面上的模式下拉菜单中找到 MissFisher 的练级模式，但该模式在出海垂钓外的行为只是使用「万能拟饵」来抛竿，然后对任何咬饵使用普通提钩。

如果你不愿意手动钓鱼，请看：用 MissFisher 出海垂钓。

否则，你可以参考攻略：NGA-捕鱼人升级路线研究与推荐。

如果你觉得 MissFisher 可以新增预设以减少在练级过程中的重复劳动，欢迎反馈。
