开始使用 ​

开始前 ​

如果没安装请先安装，如果还没认证，请先认证。

能清单式全自动吗？ ​

MissFisher 有主要针对稀有鱼的清单自动模式。但是你还必须自己准备背包空间、修装备和购买钓饵。

能干什么？ ​

钓特定鱼类 ​

可以按推荐清单自动，也可以指定预设名，是类脚本的有限自动钓鱼。

  • 钓场之王、海钓中的普通海域稀有鱼、宠物、部分捕鱼人的特殊交付需求
    （不包括老主顾交易）

以模式按场景自动 ​

可自动调用预设，有额外的检测和辅助行为。

  • 出海垂钓：
    可以进行单轮全自动以完成渔分成就、收集蓝色稀有度鱼类、完成种类数量要求的奖励成就，也可使用耐力模式进行有限的多轮全自动。（不包含精确的图鉴收集）

  • 宇宙探索：
    可以随任务接取自动启动适合当前任务的预设（提供有可设置项），不包括自动刷任务/寻路到任务钓场。

  • 云冠群岛：
    可以完成数个相关成就。（不包含精确的图鉴收集）

做不到的 ​

  • 不能高效练级
  • 不能无脑帮你点亮图鉴
  • 没有自动修理
  • 没有自动精炼
  • 没有自动精选
  • 没有自动交易大地票据
  • 没有自动购买钓饵
  • 不能自动消除钓场警惕

相关插件依赖 ​

MissFisher 依赖部分插件来提供额外功能。

可在主界面 - 标题栏的齿轮设置按钮打开设置页，找到插件依赖部分查看还未安装或启动的依赖插件。

依赖的插件MissFisher 用它来做什么
Daily Routines借用它的 Edge TTS 进行语音信息播报和提醒
vnavmesh它可以提供地图内的自动寻路 / 导航
ICE自动接取、刷取、提交宇宙探索任务
