参考 ​

这一部分收录 MissFisher / Angex 相关的正式说明，适合在“已经开始使用之后”按问题查阅。

你可以在这里查什么 ​

  • Angex 语法规范：用于核对表达式的结构、关键字和解析规则
  • 诱饵技能：用于查阅诱饵相关机制、效果和使用上的注意点

适合什么时候看 ​

  • 你已经知道大概要写什么，但想确认写法是否准确
  • 你在修改表达式时，需要核对某个符号、关键字或结构细节
  • 你在分析某个策略为什么生效，想查机制层面的说明

建议用法 ​

  • 把这里当作字典或规格说明，而不是从头连续阅读的教程
  • 遇到“这能不能这么写”“这个机制具体怎样算”时，再回来查对应页面

如果你还没开始上手 ​

如果你现在更需要的是入门路线和实际修改方法，建议先看 指南。
