## 功能

- 钓鱼王与特定鱼类预设
- 自动出海垂钓与航线辅助
- 云冠群岛与宇宙探索流程支持
- 钓法表达式与预设编辑支持

## 安装

在 Dalamud 插件安装器中添加第三方仓库链接：

```text
https://raw.githubusercontent.com/BlackCleaverLoli/MissFisher/refs/heads/main/MissFisher.json
```

完整安装、认证和使用说明请查看文档站。
