## 功能与特性

**简洁高效的自动功能、集中维护的预设数据、兼具实用和趣味性的辅助功能。**

## 安装

在 Dalamud 设置中添加第三方仓库链接：

```text
https://raw.githubusercontent.com/BlackCleaverLoli/MissFisher/refs/heads/main/MissFisher.json
```

添加完成并保存设置后，在插件安装器中搜索 `MissFisher` 并点击安装。

更详细的安装和使用说明见 [MissFisher 文档站](https://blackcleaverloli.github.io/MissFisher/)。
